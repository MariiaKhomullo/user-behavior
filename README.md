# Анализ поведения пользователей в продукте

``` Учебный проект ``` 

## Легенда

Игра Quiz Freeze — то сборник вопросов в различных форматах и темах, на которые игрокам нужно давать ответы. Игра одиночная, то есть каждый игрок отвечает на вопросы самостоятельно.

Некоторое время назад игру основательно переделали, но хороший и качественный анализ изменений, к сожалению, до сих пор не провели.

## Цель проекта

Исследовать поведение пользователей в обновлённом приложении.

## ФОРМАЛИЗОВАННЫЕ ЗАДАЧИ

Необходимо проверить у пользователей, *зарегистрировавшихся в 2018 году*:

- есть ли зависимость между выбранным уровнем сложности и вероятностью оплаты;

- различается ли временной промежуток между регистрацией и оплатой у групп пользователей с разным уровнем сложности.

## Дано
- данные о событиях, которые совершают пользователи,
- данные об оплатах, которые совершают пользователи.
